# 低压电工证刷题小工具 ⚡

> 纯前端单页 HTML 应用，深色护眼主题，离线可用。模拟真实机考体验，助力低压电工证备考。

## ✨ 功能

- **随机抽题** — 题库随机打乱，每轮不重复
- **答题卡导航** — 左侧实时答题卡，点击跳题，已答/标记/未答一目了然
- **计时考试** — 正数计时，模拟真实考试节奏
- **自动评分** — 交卷即出分，正确/错误/正确率/用时全展示
- **错题回顾** — 答错题目集中查看，附解析说明
- **键盘操作** — A/B/C/D 选答案，← → 翻题，M 键标记
- **零依赖** — 单一 HTML 文件，双击打开即用，完全离线

## 🚀 使用

直接双击 `低压电工证模拟考试.html` 即可在浏览器中打开使用。

## 🛠 技术

纯 HTML + CSS + Vanilla JavaScript，内嵌题库于 `<script>` 标签，无外部依赖。

## 📝 题库

初始题库涵盖低压电工证考试核心知识点：配电系统、接地保护、触电防护、电气安全等。欢迎贡献题目或自行扩充 `QUESTION_BANK` 数组。
